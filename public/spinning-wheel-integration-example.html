How It Works

This page demonstrates how to integrate the spinning wheel modal into your existing website. The modal will automatically appear after 5 minutes of browsing, but only once per day per user.

Integration Steps

  1. Include the script in your HTML pages:
    2. <script src="js/spinning-wheel-trigger.js"></script>
  2. That's it! The modal will automatically appear after 5 minutes.

Customization Options

You can customize the behavior by modifying the configuration in the script:

const CONFIG = { TRIGGER_DELAY: 5 * 60 * 1000, // Change to 3 minutes: 3 * 60 * 1000 STORAGE_KEY: 'spinningWheelLastSeen', MODAL_ID: 'spinningWheelModal', IFRAME_SRC: 'spinning-wheel-standalone.html', Z_INDEX: 9999 };

Manual Control

You can also manually control the modal using these global functions:

// Show modal immediately window.showSpinningWheelModal(); // Close modal window.closeSpinningWheelModal();
